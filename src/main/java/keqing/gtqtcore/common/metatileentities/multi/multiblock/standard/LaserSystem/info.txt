1.高能激光系统
原料：氙离子激光器
涉及产线：氙线 超能硅岩
激光发射器：主体为一个需要电的多方块，供能后其正前方将发射出一束激光(不能拐弯)
激光集散器：合并或者分离激光，六面可控
激光接收器：接受激光将其还原到电力能源
本项目将作为gtceu自带激光能源前置研究

2.新聚变系统激光点火器(比较抽象)
聚变的没有开机税版本，需要激光功率达到阈值即可开机(不能断激光)

3.激光加热器接口＆激光类多方块
加热接口为激光类多方块的一个仓， 激光类多方块包括：
激光电炉
激光高炉
激光合金炉

注：此类多方块无需线圈便可运行基于当前激光器等级的加热配方

激光切割机(切割机)
激光离心机(热能离心 离心)

以上多方块：并行与耗时减免基于距离与激光强度计算，基于激光强度计算电压，多方块不耗电。在激光器点火后，温度会上升到最大值，机器运行速度也会越来快。

使用 激光发射器 对准加热接口即可，不需要贴着，但是距离也不要太远(8格以内)

4.超远距离激光无线电-天基棱镜系统
使用激光发射器将激光发送到聚焦器内，即可定位指定坐标(当前纬度)的接收器内，时刻传输激光能量
